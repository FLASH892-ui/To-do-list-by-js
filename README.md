# To-Do List Project

This repository contains two versions of a To-Do List application.

## 1. Vanilla JavaScript
The original implementation using HTML, CSS, and Vanilla JavaScript.

-   **Source Code**: [`/default-langs`](./default-langs)
-   **Live Demo**: [https://to-do-list-by-js-mu.vercel.app](https://to-do-list-by-js-mu.vercel.app)

## 2. Vite + React + TypeScript + Tailwind CSS
A modern rewrite of the application using React and Tailwind CSS.

-   **Source Code**: [`/vite-react-ts-tailwind`](./vite-react-ts-tailwind)
-   **Live Demo**: [https://FLASH892-ui.github.io/To-do-list-by-js/](https://FLASH892-ui.github.io/To-do-list-by-js/)

---

### How to Run Locally

#### Vanilla JS
Simply open `default-langs/index.html` in your browser.

#### React + Vite
```bash
cd vite-react-ts-tailwind
npm install
npm run dev
```
